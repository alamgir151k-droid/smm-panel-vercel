# SMM Panel - Vercel Ready

یہ پروجیکٹ Vercel پر لائیو کرنے کے لیے تیار کیا گیا ہے۔

## GitHub پر اپلوڈ کرنے کا طریقہ:
1. ایک نیا GitHub ریپوزٹری بنائیں۔
2. ان تمام فائلوں کو وہاں اپلوڈ کریں۔

## Vercel پر لائیو کرنے کا طریقہ:
1. Vercel پر جائیں اور اپنی GitHub ریپوزٹری کو امپورٹ کریں۔
2. **Environment Variables** میں درج ذیل ویلیوز شامل کریں (اپنے ڈیٹا بیس کے مطابق):
   - `DB_HOST`: آپ کے ڈیٹا بیس کا ہوسٹ (مثلاً: `localhost` یا ریموٹ ہوسٹ)
   - `DB_NAME`: ڈیٹا بیس کا نام
   - `DB_USER`: ڈیٹا بیس کا یوزر نیم
   - `DB_PASS`: ڈیٹا بیس کا پاس ورڈ

## ڈیٹا بیس سیٹ اپ:
- `database.sql` فائل کو اپنے MySQL ڈیٹا بیس میں امپورٹ کریں۔

## اہم نوٹ:
- Vercel پر PHP چلانے کے لیے `vercel-php` رن ٹائم استعمال کیا گیا ہے جو `vercel.json` میں کنفیگر ہے۔
- `app/config.php` اب خود بخود آپ کے ڈومین اور ڈیٹا بیس کی معلومات انوائرمنٹ ویری ایبلز سے اٹھا لے گا۔
